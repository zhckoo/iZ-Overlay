# iZ-Overlay
Real-time telemetry overlay for iRacing using Electron.js

# iZ Overlay (In-Progress) 🏎️

iRacing simülasyonu için modern web teknolojileriyle geliştirilen, gerçek zamanlı telemetri veri görselleştirme aracı.

## 🚀 Proje Amacı
Bu proje, yarış esnasında sürücülerin ihtiyaç duyduğu kritik verileri (hız, RPM, lastik sıcaklığı, yakıt durumu vb.) iRacing SDK üzerinden çekerek özelleştirilebilir bir HUD (Heads-up Display) katmanında sunmayı hedefler.

## 🛠️ Kullanılan Teknolojiler
* **Electron.js:** Masaüstü uygulama mimarisi.
* **JavaScript (Node.js):** Backend veri işleme ve SDK entegrasyonu.
* **HTML5 & CSS3:** Kullanıcı arayüzü ve animasyonlar.

# iZ Overlay (In-Progress) 🏎️

A high-performance, real-time telemetry visualization tool for iRacing, built with modern web technologies and Electron.js.

## 🚀 Project Goal
The primary objective of this project is to extract critical real-time telemetry data (Speed, RPM, Tire Temperatures, Fuel Status, etc.) via the **iRacing SDK** and present it through a fully customizable, low-latency **HUD (Heads-up Display)**. It aims to bridge the gap between hobbyist sim racing and professional data visualization.

## 🛠️ Tech Stack
* **Electron.js:** Desktop application architecture and window management.
* **JavaScript (Node.js):** Backend logic and SDK integration middleware.
* **HTML5 & CSS3:** Responsive user interface and real-time animations.
